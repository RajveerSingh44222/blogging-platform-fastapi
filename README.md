# 📝 Blogging Platform

A full-stack blogging platform built using **FastAPI, MySQL, and JavaScript**, enabling users to create, explore, and manage blogs efficiently.

---

## 🚀 Features

- ✍️ Create, Read, Update, Delete (CRUD) Blogs
- 🌐 Explore Blogs Page
- 🏷️ Tag-Based Filtering
- 🔖 Save/Bookmark Blogs
- ⚡ FastAPI REST API Integration
- 🎯 Clean and responsive UI

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- FastAPI
- Python

**Database**
- MySQL

---

## ⚙️ How to Run Locally

### 1. Clone the Repository
git clone https://github.com/RajveerSingh44222/blogging-platform-fastapi.git
cd blogging-platform-fastapi

### 2. Backend Setup
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### 3. Database Setup
Create a MySQL database and update connection in backend/database.py

### 4. Frontend
Open frontend/index.html in browser
