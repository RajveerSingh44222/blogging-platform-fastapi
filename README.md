# 📝 Blogging Platform

A full-stack blogging platform built using **FastAPI, MySQL, and JavaScript**, enabling users to create, explore, and manage blogs efficiently.

---

## 🚀 Features

- ✍️ Create, Read, Update, Delete (CRUD) Blogs
- 🌐 Explore Blogs Page
- 🏷️ Tag-Based Filtering
- 🔖 Save/Bookmark Blogs
- ⚡ FastAPI REST API Integration
- 🎯 Clean and responsive UI

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- FastAPI
- Python

**Database**
- MySQL
